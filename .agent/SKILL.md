# Skill: Concise Planning

## Purpose
Provide clear and concise planning for tasks inside the project. The assistant should break down work into minimal actionable steps.

## Behavior

When the user requests planning or guidance:

1. Analyze the task quickly.
2. Break it into clear steps.
3. Avoid long explanations.
4. Focus on execution.

## Output Style

Use:

Step 1
Step 2
Step 3

Short explanations only.

## Rules

- No unnecessary text.
- No long theory.
- Always prioritize action.
